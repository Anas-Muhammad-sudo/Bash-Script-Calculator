# AnasCalc - Bash Script Calculator

## Overview
AnasCalc is a simple calculator written in Bash. It performs basic arithmetic operations such as multiplication, division, addition, and subtraction. The script reads two numbers from the user and performs the selected operation.

## Files in the Repository
- **calc**: This is the main script that handles user input and performs the operations.
- **mult**: A script containing the multiplication function used by the main script.

## Usage
1. Clone the repository to your local machine.
2. Ensure the `calc` and `mult` scripts have executable permissions:
   ```bash
   chmod +x calc mult
3. Run APP using ./calc.sh on linux
